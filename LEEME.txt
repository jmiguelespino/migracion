REQUISITOS
──────────
• Python 3 (ya viene instalado en Windows 10/11, macOS y Linux)
• Según el motor de IA que elijas:
  - Claude .......... API key de Anthropic (console.anthropic.com) + internet
  - 🆓 Gratis ....... Ollama instalado (ollama.com) + un modelo descargado.
                      NO necesita API key, internet ni tokens: corre local.
                      Recomendado (rápido en CPU):
                        ollama pull qwen2.5-coder:1.5b
                      Más calidad pero lento sin GPU:
                        ollama pull qwen2.5-coder

  La "App completa (cobertura total)" NO requiere ningún motor: funciona
  siempre, sin API key ni Ollama.


CÓMO INICIAR
────────────
Windows:
  1. Hacé doble clic en:  INICIAR.bat

macOS / Linux:
  1. Abrí una terminal en esta carpeta
  2. Ejecutá:  bash iniciar.sh

(o en cualquier sistema:  python servidor.py)

  → Se abre una ventana del servidor — NO la cierres
  → Abrí tu navegador en:  http://localhost:8080


CÓMO USAR EL AGENTE
────────────────────
1. Elegí el motor de IA en "Motor de IA":
   • Claude  → pegá tu API key de Anthropic y hacé clic en OK
   • Gratis  → elegí el modelo local (necesitás Ollama corriendo)
   (La "App completa" no necesita motor.)
2. Subí el ZIP con el sistema legacy
3. Elegí el lenguaje origen y la tecnología destino
4. Hacé clic en "Analizar sistema"
   → El agente LEE el ZIP de verdad: cuenta los formularios,
     reportes y programas, y extrae la estructura real de las
     tablas .dbf (campos y tipos). El análisis ya no es inventado.
5. Seleccioná una fase para generarla
6. Descargá la fase: baja un ZIP con cada archivo en la carpeta src/,
   los tests en tests/ y un README.md con las instrucciones


DÓNDE OBTENER LA API KEY
─────────────────────────
1. Entrá a https://console.anthropic.com
2. Creá una cuenta
3. Menú → API Keys → Create Key
4. Copiá la clave (empieza con sk-ant-)


PARA CERRAR
───────────
Cerrá la ventana del servidor o presioná Ctrl+C en ella.


SOPORTE
───────
Si aparece error de conexión:
• Verificá que la ventana del servidor esté abierta
• Verificá que la URL sea http://localhost:8080
• Verificá tu API key en console.anthropic.com (motor Claude)

Modo gratuito (Ollama):
• Instalá Ollama desde https://ollama.com y abrilo (debe quedar corriendo)
• Descargá un modelo de código:  ollama pull qwen2.5-coder:1.5b
• La lista de modelos aparece sola en la barra lateral al elegir "Gratis"
• Los modelos locales pueden tardar más que Claude según tu PC
• Si el análisis "se queda colgado" o da timeout: estás usando un modelo muy
  pesado para tu CPU. Usá el 1.5b (ollama pull qwen2.5-coder:1.5b) y elegilo
  en la lista de modelos.

Si una fase "se corta por longitud": volvé a intentar. El agente
ahora usa max_tokens alto, pero fases muy grandes pueden requerir
reintentar o dividirse.
